About Datasets2Tools
Datasets2Tools is a platform for the discovery and evaluation of biomedical digital objects. Datasets2Tools indexes over 30,000 bioinformatics analyses, over 6,000 biological datasets and over 6,000 computational tools. Users can find relevant objects by using the search interface, the Google Chrome extension and the Application Programming Interface (API). Several metrics are integrated in the search results to help users find the most relevant results. These include PubMed citations PlumX metrics, Altmetric Attention Scores, and the results of FAIR evaluations by the crowd.
The Canned Analysis
The Canned Analysis is a new type of digital object designed to store the results of bioinformatics analyses. It is designed to index such objects in a findable, accessible, interoperable and reusable manner. Canned analyses are defined by three key elements: references to the datasets and tools used to generate it, text annotations, and metadata.
The FAIR Principles
The FAIR Principles are a set of high-level principles which describe characteristics which digital objects should have to promote findable, accessible, interoperable and reusable data management.
