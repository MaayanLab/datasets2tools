General Information

What is Datasets2Tools?

Datasets2Tools is a platform which aims to promote findable, accessible and shareable bioinformatics analyses of biomedical data. It is comprised of three parts: a database of dataset-tool associations, a web platform, and a search engine.

What is a Canned Analysis?

A canned analysis is a pre-run analysis of a biomedical dataset by a computational tool, whose results can be accessed and interactively browsed on a webpage identified by a persistent URL.

What is the Datasets2Tools Chrome Extension?

The Datasets2Tools Chrome extension enriches biomedical repositories such as GEO and DataMed.org by embedding links to canned analyses to dataset landing pages and dataset search pages.

Search

How do I use the keyword search?

To use the keyword search functionality, begin by selecting the type of object you want to find - analyses, datasets or tools. Then insert one or more terms relevant to what you would like to find, separating them by using the Enter key. Terms can contain spaces, numbers and symbols. These will be used as keywords to filter the objects you want to find based on their metadata.

How do I use the advanced search form?

To use the advanced search form, begin by selecting the type of object you want to find - analyses, datasets or tools. Proceed by selecting which metadata field you want to filter on, the type of filter, and then insert the search term in the input text box. To add more search terms to your query, click on the plus icon on the right of the first input row. To remove undesired search terms, click on the minus button on the right of the additional input rows.

How do I search using the API?

Contribute

How do I add new analyses, datasets and tools to Datasets2Tools?

Datasets2Tools currently provides two methods to upload analyses to our database:
  • A form to manually upload analyses
  • An API to upload analyses in bulk

API

How do I use the API to find objects?

Datasets2Tools provides an API to easily find analyses, datasets and tools of interest.
There are three endpoints:
  • https://amp.pharm.mssm.edu/datasets2tools/api/analysis, to find canned analyses.
  • https://amp.pharm.mssm.edu/datasets2tools/api/dataset, to find datasets.
  • https://amp.pharm.mssm.edu/datasets2tools/api/tool, to find tools.
For each query, the API returns a JSON object containing information relevant to the objects queried.
For detailed usage information, visit the manual section for the analysis, dataset and tool endpoints below.

Finding analyses: the analysis endpoint

The analysis endpoint allows to search canned analyses by filtering on the associated metadata, datasets and tools.
Examples:
How do I find analyses associated to dataset GSE37618?
http://amp.pharm.mssm.edu/datasets2tools/api/analysis?dataset_accession=GSE37618
How do I find analyses generated by Enrichr?
http://amp.pharm.mssm.edu/datasets2tools/api/analysis?tool_name=Enrichr
How do I find analyses generated by Enrichr on GSE37618?
http://amp.pharm.mssm.edu/datasets2tools/api/analysis?tool_name=Enrichr&dataset_accession=GSE37618
How do I find analyses relevant to prostate cancer?
http://amp.pharm.mssm.edu/datasets2tools/api/analysis?disease_name=prostate cancer
How do I find analyses relevant to ESR1 KD experiments?
http://amp.pharm.mssm.edu/datasets2tools/api/analysis?hs_gene_symbol=ESR1&pert_type=KD

To perform more advanced queries, please read the advanced search API manual here.

Find datasets: the dataset endpoint

The dataset endpoint allows to search datasets by filtering on the associated metadata.
Examples:
How do I find dataset GSE37618?
http://amp.pharm.mssm.edu/datasets2tools/api/dataset?dataset_accession=GSE37618
How do I find datasets from the LINCS repository?
http://amp.pharm.mssm.edu/datasets2tools/api/dataset?repository_name=LINCS
The dataset endpoint currently supports searching by dataset accession and repository. Additional functionalities soon to come.
To perform more advanced queries, please read the advanced search API manual here.

Find tools: the tool endpoint

The tool endpoint allows to search tools by filtering on the associated metadata.
Examples:
How do I find information regarding the tool Enrichr?
http://amp.pharm.mssm.edu/datasets2tools/api/tool?tool_name=Enrichr
The dataset endpoint currently supports searching by tool name. Additional functionalities soon to come.
To perform more advanced queries, please read the advanced search API manual here.

Extension

What is the Datasets2Tools Chrome Extension?

The Datasets2Tools Chrome extension enriches biomedical repositories such as GEO and DataMed.org by embedding links to canned analyses to dataset landing pages and dataset search pages.

Where is the Chrome Extension active?

The Datasets2Tools Chrome extension is currently active on dataset landing pages and dataset search pages of the Gene Expression Omnibus and DataMed.org.
You can test out the extension at the following links: GEO Landing, DataMed Landing, GEO Search, DataMed Search.
